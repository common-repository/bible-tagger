This plugin enables Bible Tagger on your WordPress site

== Description ==

A WordPress plugin based on the NETBible API, that allows the bible references on your pages to popup when one mouse overs the reference. see to https://labs.bible.org/NETBibleTagger/

== Installation ==

1. Upload the entire `bible-tagger` folder to your `/wp-content/plugins/` folder.
2. Go to the 'Plugins' page in the menu and activate the plugin.
3. Navigate to Settings -> Bible Tagger in your WordPress dashboard to customise the settings.

==Plugin Features/options==

1. This plugin enables popup tagger options in response to the APIs from Bible.org.
2. The plugin is compatible with the latest version of Wordpress.
3. The plugin allows users to tag verses , references in their post.
4. The tagger accepts the references tags from posts and verifies with APIs from bible.org.
5. Then the plugin displays the references which is defined in the posts in a pop form.
6. A user can customise the fonts sizes, add custom css and modify the text content.
